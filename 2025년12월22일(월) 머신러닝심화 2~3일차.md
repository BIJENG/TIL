### 2️⃣ 회귀 평가 지표 정리

#### ✔ MSE (Mean Squared Error)
- 오차를 제곱해서 평균
- 큰 오차에 민감

#### ✔ RMSE
- MSE에 루트를 씌운 값
- 단위가 원래 값과 같아 해석 쉬움

#### ✔ R² (결정계수)
- 모델이 데이터를 얼마나 설명하는지
- 1에 가까울수록 좋음

8️⃣ Lasso vs LassoCV
| 구분       | Lasso          | LassoCV |
| -------- | -------------- | ------- |
| alpha 선택 | 수동             | 자동      |
| 교차검증     | ❌              | ⭕       |
| 목적       | 일반 모델          | 편의용     |
| 실무       | GridSearch와 함께 | 빠른 실험   |
📌 LassoCV는 alpha 전용 자동 도구
📌 GridSearch는 모든 모델에 적용 가능한 일반 방법


2. 심화 3일차 초반 – SVM (Support Vector Machine)

1️⃣ SVM은 어떤 모델인가?
분류 모델
데이터를 나누는 경계선을 찾음
단순히 나누는 게 아니라 가장 안전한 경계선

2️⃣ 마진(Margin)
경계선과 가장 가까운 데이터까지의 거리
SVM의 목표는 마진 최대화
📌 마진이 크다 =
👉 데이터가 잘 분리됨
👉 새로운 데이터에도 안정적

3️⃣ 서포트 벡터(Support Vector)
경계선에 가장 가까이 붙어 있는 데이터들
이 점들이 경계선을 결정
👉 모든 데이터가 아니라
👉 일부 핵심 데이터만 중요

4️⃣ SVM과 스케일링
SVM은 거리 기반 모델
✔ StandardScaler 거의 필수
❌ 스케일링 안 하면 성능 급락

5️⃣ SVM과 해석
가중치 해석 ❌
feature_importances_ ❌
성능 중심 모델
📌 SVM은
“왜 그렇게 판단했는가”보다
“얼마나 잘 나누는가”에 집중
