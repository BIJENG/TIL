📘 기초 수학 & NumPy 기초 정리

AI가 헬스케어 데이터를 다루는 방식의 핵심은 데이터를 수학적으로 표현하는 것이다.
즉, 복잡한 환자 정보도 결국 AI가 이해할 수 있는 형태인 스칼라(Scalar), 벡터(Vector), 행렬(Matrix) 로 변환된다.

🔢 1. AI가 데이터를 보는 방식

AI는 세상을 다음과 같은 수학적 단위로 이해한다:

✔ 스칼라 (Scalar)

하나의 숫자로 표현되는 가장 작은 단위의 데이터
예시:

환자 체온: 36.5

혈당 수치: 96

✔ 벡터 (Vector)

하나의 대상(환자 1명)을 설명하는 여러 개의 숫자(스칼라)를 순서 있게 나열한 것 → 1차원 배열

예시 – 환자 김오즈의 건강 데이터:

kimoz = [60, 146, 77.4, 32]  
# [운동 시간(분), 최대 심박수, 체중(kg), 나이(세)]


벡터를 사용할 때 중요한 것은 각 인덱스에 어떤 데이터가 들어가는지 통일된 약속을 유지하는 것.

✔ 행렬 (Matrix)

여러 사람(벡터)을 한데 모아 구성한 2차원 배열

예:

[환자1 벡터]
[환자2 벡터]
[환자3 벡터]

🧩 3. NumPy 기본

NumPy는 파이썬에서 수치 연산을 고속으로 수행하기 위해 사용하는 대표 라이브러리다.

✔ NumPy 불러오기
import numpy as np


VSCode 또는 로컬 환경에서는 pip install numpy 설치 필요.
Google Colab은 기본 탑재되어 있음.

📏 4. NumPy 배열 만들기
1차원 배열
list_1d = [1, 2, 3, 4, 5]
arr1d = np.array(list_1d)
print(arr1d)

2차원 배열 (행렬)
list_2d = [[10, 20, 30], [40, 50, 60]]
arr2d = np.array(list_2d)
print(arr2d)

🔍 5. 배열의 속성 확인
속성	설명
.shape	배열의 형태(행, 열)
.ndim	차원의 수
.dtype	데이터 타입

예:

print(arr2d.shape)  # (2, 3)
print(arr2d.ndim)   # 2
print(arr2d.dtype)  # int32 등

🎯 6. 인덱싱 & 슬라이싱
✔ 기본 인덱싱
arr[2]      # 3번째 행 또는 원소
arr[2, 1]   # 3행 2열

✔ 슬라이싱
arr[:, 1]       # 모든 행의 2열 값
arr[0:2, :]     # 1~2번째 행 전체

⚡ 7. 벡터화 연산 (Vectorized Operation)

NumPy의 핵심 기능: 반복문 없이 배열 전체에 연산 수행

✔ 스칼라 vs 벡터
arr = np.array([1, 2, 3])
arr + 10    # → [11, 12, 13]
arr * 2     # → [2, 4, 6]

✔ 벡터 vs 벡터
mid = np.array([80, 75, 90])
final = np.array([85, 95, 88])

total = mid + final
# → [165, 170, 178]

🔍 8. 조건 필터링 (Boolean Indexing)

조건을 만족하는 데이터만 골라낼 수 있음.

age_arr = np.array([10, 45, 22, 58, 31])
mask = age_arr > 30

age_arr[mask]   # → [45, 58, 31]

📊 9. 기본 통계 함수

np.mean() → 평균

np.max() → 최댓값

np.min() → 최솟값

np.sum() → 합계

np.std() → 표준편차

예:

np.mean(arr)
np.std(arr)

🧠 11. 가중합(Weighted Sum) — AI 예측의 기본 원리

AI 모델은 여러 특성(입력값)에 가중치를 곱해 위험도를 계산한다.

✔ 예시: 당뇨병 위험도 계산

환자 A 데이터 (벡터)

[혈당 150, 체중 92, 나이 58]


학습된 가중치 벡터

[0.7, 0.2, 0.1]

직접 계산
150×0.7 + 92×0.2 + 58×0.1


NumPy로 계산하면:

patient = np.array([150, 92, 58])
weights = np.array([0.7, 0.2, 0.1])

score = np.sum(patient * weights)

📘 12. 벡터 vs NumPy 배열 — 무엇이 다른가?
구분	벡터(Vector)	NumPy 배열(ndarray)
의미	수학적 개념	프로그래밍 구현
구조	1차원만 존재	N차원 확장 가능
연산	수학 공식 중심	고속 벡터·행렬 연산
생성	리스트 등	np.array()

✔ 결론:
NumPy 배열을 사용해 벡터를 ‘프로그래밍적으로’ 표현하는 것

📚 정리

오늘 배운 핵심:

AI는 데이터를 스칼라 → 벡터 → 행렬 구조로 이해한다.
Python에서는 이를 NumPy 배열로 표현한다.
벡터화 연산, 전치, 조건 필터링 등은 AI 데이터 처리의 핵심 기술이다.
가중합(weighted sum)은 AI 모델 예측의 기본 원리다.
