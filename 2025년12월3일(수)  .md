# MySQL 학습 회고

## 1. 집계 함수(Aggregation Functions)

| 함수 | 설명 |
|------|------|
| `COUNT()` | 행 개수 |
| `SUM()` | 합계 |
| `AVG()` | 평균 |
| `MIN()` / `MAX()` | 최소값 / 최대값 |
| `STDDEV()` | 표준편차 |

> ⚠️ 집계 함수는 `NULL` 값을 자동으로 제외하고 계산합니다.

## 4. 데이터베이스 생성 & 삭제

```sql
-- 생성
CREATE DATABASE db_name;

-- 사용
USE db_name;

-- 삭제
DROP DATABASE db_name;
Workbench에서는 여러 DB 존재 가능, USE로 작업 대상 DB 선택 필요

5. 주석 사용법
sql
코드 복사
-- 단일 주석
/* 여러 줄 주석 */
실행 시 드래그 후 번개 버튼 클릭 → 선택 영역만 실행 가능

6. 테이블 관리
sql
코드 복사
-- 테이블 생성
CREATE TABLE table_name (
    column1 datatype constraints,
    column2 datatype constraints,
    FOREIGN KEY (patient_id) REFERENCES patients(patient_id)
);

-- 테이블 삭제
DROP TABLE table_name;

-- 테이블 구조 확인
DESCRIBE table_name;

-- 테이블 목록 확인
SHOW TABLES;
ALTER TABLE 사용법
sql
코드 복사
-- 컬럼 추가
ALTER TABLE table_name ADD COLUMN new_column_name datatype;

-- 컬럼 삭제
ALTER TABLE table_name DROP COLUMN column_name;

-- 컬럼 수정
ALTER TABLE table_name MODIFY column_name datatype;
ENUM 사용 예시: ENUM('M', 'F') → 'M' 또는 'F'만 입력 가능

7. 데이터 조작
INSERT
sql
코드 복사
-- 단일 행 추가
INSERT INTO table_name(col1, col2, col3)
VALUES ('값1', '값2', '값3');

-- 여러 행 추가
INSERT INTO table_name(col1, col2, col3)
VALUES 
    ('값1', '값2', '값3'),
    ('값4', '값5', '값6'),
    ('값7', '값8', '값9');
컬럼 순서는 자유롭게 지정 가능

VALUES 안에서 여러 행 추가 가능

UPDATE
sql
코드 복사
UPDATE table_name
SET column1 = value1, column2 = value2
WHERE 조건;
⚠️ WHERE 없이 실행하면 전체 데이터가 변경됨

DELETE
sql
코드 복사
DELETE FROM table_name
WHERE 조건;
⚠️ 헬스케어 시스템에서는 데이터 삭제 거의 금지 → soft delete 사용 권장
