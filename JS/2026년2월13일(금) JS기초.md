# 📚 JavaScript 기초 문법 정리 (변수 ~ 조건문)

오늘 학습한 자바스크립트의 핵심 내용을 정리했습니다. 파이썬과 비슷하면서도 다른 자바스크립트의 특징에 주목해 보세요.

---

## 1. JavaScript 실행 방법

- **브라우저 콘솔**: `F12` 개발자 도구에서 즉시 실행.
- **Node.js**: 터미널에서 `node 파일명.js`로 실행 (서버 환경).
- **HTML 연결**: `<script src="app.js"></script>`를 통해 웹 페이지에 적용.

## 2. 변수 선언 (`let` vs `const`)

자바스크립트는 파이썬과 달리 변수 선언 시 키워드를 사용합니다.

- **`let`**: 값이 바뀔 수 있는 변수 (Mutable).
- **`const`**: 한 번 정하면 바꿀 수 없는 상수 (Immutable). **기본적으로 `const`를 쓰고, 필요할 때만 `let`으로 바꾸는 것이 관습입니다.**

```javascript
let score = 750; // 변경 가능
const apiKey = "medical-ai-token"; // 변경 불가능
```

## 3. 자료형 (Data Types)

- **Number**: 정수와 실수 구분 없음.
- **String**: 따옴표로 감싼 문자열.
- **Boolean**: `true`, `false` (소문자 주의!).
- **null**: "비어 있음"을 의도적으로 명시.
- **undefined**: 값이 아직 할당되지 않은 상태.

## 4. 연산자 (Operators)

### 일치 연산자 (중요!)

- `==`: 값만 비교 (타입이 다르면 자동 변환함. 권장하지 않음).
- `===`: **값과 타입이 모두 같아야 true** (엄격한 비교. 권장함).

```javascript
10 == "10"; // true
10 === "10"; // false
```

## 5. 조건문 (`if` / `else if` / `else`)

파이썬의 `if-elif-else`와 구조가 같습니다. 다만 조건식을 `()`로 감싸고 실행 범위를 `{}`로 나눕니다.

```javascript
const score = 95;

if (score >= 90) {
  console.log("A 등급 - 의료 AI 전문가!");
} else if (score >= 70) {
  console.log("B 등급 - 훌륭합니다.");
} else {
  console.log("C 등급 - 조금 더 연습해봅시다.");
}
```

## 💡 의료 AI 지망생을 위한 보충 팁: Truthy & Falsy

자바스크립트에서는 `if`문 조건에 꼭 boolean이 아니더라도 **값의 존재 여부**에 따라 참/거짓으로 판정합니다.

- **Falsy (거짓으로 취급)**: `false`, `0`, `""`(빈 문자열), `null`, `undefined`, `NaN`.
- **Truthy (참으로 취급)**: 위 6개를 제외한 모든 것 (빈 배열 `[]`, 빈 객체 `{}` 포함).

> **실전 예시**: 데이터 분석 중 `if (medicalData)`라고 쓰면, 데이터가 비어있지 않을 때만 로직을 실행하게 됩니다.
