# 📘 JavaScript 핵심 복습 (Array ~ Event Listener)

---

## 1. 배열 (Array) & 반복문

데이터가 많아지면 변수 하나하나에 담을 수 없습니다. 리스트 형태로 관리합니다.

### ✅ 핵심 문법

- **선언**: 대괄호 `[]` 사용
- **접근**: 인덱스(0부터 시작) 사용

```javascript
// 과일 목록 선언
const fruits = ["사과", "바나나", "포도"];

// 데이터 꺼내기
console.log(fruits[0]); // "사과"
console.log(fruits.length); // 3 (개수 확인)
```

### 🔄 데이터 꺼내 쓰기 (반복문)

배열 안에 있는 모든 데이터를 화면에 보여주거나 계산할 때 사용합니다.

```javascript
// 1. 기본 for문 (인덱스가 필요할 때)
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// 2. for...of문 (요즘 많이 쓰는 방식, 값만 필요할 때)
for (const fruit of fruits) {
  console.log(fruit);
}
```

## 2. DOM (Document Object Model)

HTML 태그를 자바스크립트가 알아먹을 수 있는 **객체(Object)**로 바꾼 것입니다.
"HTML을 조작하기 위한 지도"라고 생각하세요.

### 🔍 요소 찾기 (선택)

가장 많이 쓰는 만능 도구는 `querySelector`입니다.

```javascript
// id로 찾기 (#)
const title = document.querySelector("#main-title");

// class로 찾기 (.)
const buttons = document.querySelector(".btn-submit");

// 태그로 찾기
const input = document.querySelector("input");
```

### ✏️ 내용 & 스타일 바꾸기 (조작)

찾은 요소(변수)를 통해 HTML을 변경합니다.

```javascript
// 텍스트 변경 (가장 많이 씀)
title.textContent = "반갑습니다!";

// 스타일 변경 (CSS 속성은 카멜케이스 사용: background-color -> backgroundColor)
title.style.color = "blue";
title.style.fontSize = "20px";
```

## 3. 이벤트 (Event) & 리스너

정적인 웹 페이지에 생명을 불어넣는 단계입니다.
**"어떤 행동(Event)을 하면, 이 함수(Function)를 실행해라!"**

### 👂 기본 구조 (addEventListener)

```javascript
// 요소.addEventListener("이벤트종류", 실행할함수);

const btn = document.querySelector("#save-btn");

btn.addEventListener("click", function () {
  console.log("버튼이 클릭되었습니다!");
  // 여기에 동작할 코드를 넣습니다.
});
```

### ⚡ 자주 쓰는 이벤트 종류

1. **click**: 요소를 마우스로 클릭했을 때 (버튼 등)
2. **input**: 사용자가 입력창에 무언가 칠 때 실시간 감지 (검색창 등)
3. **submit**: 폼 데이터를 전송할 때

## 4. 실전 패턴: 입력값 가져와서 화면에 뿌리기

오늘 배운 모든 것을 합친 **가장 중요한 패턴**입니다. (To-Do 리스트, 댓글 기능의 기초)

```javascript
// 1. HTML 요소 가져오기
const input = document.querySelector("#username");
const button = document.querySelector("#register");
const display = document.querySelector("#result");

// 2. 이벤트 연결하기
button.addEventListener("click", function () {
  // 3. 입력값 읽기 (.value)
  const text = input.value;

  // 유효성 검사 (빈 값 방지)
  if (text === "") {
    alert("이름을 입력해주세요!");
    return;
  }

  // 4. 화면에 출력하기 (.textContent)
  display.textContent = `환영합니다, ${text}님!`;

  // (선택) 입력창 비우기
  input.value = "";
});
```

---

## 💡 Tips

1. **`querySelector`는 오타에 민감합니다.** `null` 에러가 나면 HTML의 id/class 철자를 꼭 확인하세요.
2. **`input` 태그의 값은 항상 `value`로 가져옵니다.** (`textContent` 아님!)
3. **`const`를 기본으로 쓰세요.** 값이 변하는 변수만 `let`으로 바꾸는 습관이 에러를 줄여줍니다.
