# 🎨 CSS 핵심 요약
> 
## 1️⃣ 박스 모델 & Display (기본기)

### (1) 🧱 `display: block` (벽돌 모드)
- **특징**: 혼자서 한 줄을 통째로 차지함 (독점).
- **크기**: `width`, `height`, `margin`, `padding` 모두 완벽 적용.
- **태그**: `<div>`, `<p>`, `<h1>`, `<ul>`, `<li>`

### (2) 🌊 `display: inline` (흐르는 물 모드)
- **특징**: 텍스트처럼 옆으로 흐름. 콘텐츠 크기만큼만 공간 차지.
- **주의**: **상하(Top/Bottom) Margin 적용 불가!** (좌우는 가능)
- **태그**: `<span>`, `<a>`, `<b>`, `<strong>`

### (3) 🎁 `display: inline-block` (하이브리드 모드)
- **특징**: 옆으로 흐르지만(Inline), 박스 크기는 가짐(Block).
- **활용**: 메뉴바 버튼, 카드 리스트 나열 시 사용.
- **팁**: 엔터(줄바꿈)가 미세한 공백(4px)을 만드니 주의할 것.

### (4) 📐 Box Sizing (크기 계산법)
- `content-box` (기본값): 테두리/패딩 주면 박스가 커짐 (**비추천**).
- `border-box` (**국룰**): 테두리/패딩을 줘도 지정한 Width 안에서 해결 (**추천**).

## 2️⃣ 위치 잡기: Position 속성

### (1) 📌 `static` (기본값)
- 문서의 흐름대로 배치. (`top`, `left` 좌표 안 먹힘)

### (2) 👻 `relative` (상대적 위치 - 유령의 집)
- **기준**: '원래 자신이 있어야 할 자리'
- **특징**: 이동해도 원래 자리는 투명하게 남아있음 (공간 차지).
- **용도**: 미세 조정 하거나, `absolute` 자식의 **'기준점(부모)'** 역할.

### (3) 🦅 `absolute` (절대적 위치 - 독수리)
- **기준**: `static`이 아닌 가장 가까운 조상(부모). 없으면 화면 전체.
- **특징**: 공중에 붕 떠서 다른 요소 위를 덮음 (원래 자리 사라짐).
- **공식**: "부모는 `relative`, 자식은 `absolute`"

### (4) 📍 `fixed` (화면 고정)
- **기준**: 브라우저 화면(Viewport).
- **특징**: 스크롤을 내려도 화면 그 위치에 딱 붙어있음.
- **예시**: 네비게이션 바, '맨 위로' 버튼, 광고 배너.

## 3️⃣ 레이아웃의 꽃: Flexbox (1차원 배치)

### [ 부모(Container)에게 주는 명령 ]

#### 👉 `flex-direction` (진행 방향)
- `row` (기본): 왼쪽 ➡ 오른쪽
- `column`: 위 ⬇ 아래
- `*-reverse`: 반대 방향 (오➡왼, 아래➡위)

#### 👉 `justify-content` (메인축 정렬 - 가로)
- `flex-start` / `center` / `flex-end`
- `space-between`: 양 끝 벽에 붙이고 간격 균등 (**가장 많이 씀**)
- `space-around`: 각 아이템 둘레에 균등한 여백

#### 👉 `align-items` (교차축 정렬 - 세로, 한 줄일 때)
- `stretch` (기본): 쭉 늘림
- `center`: 세로 중앙 정렬 (**가장 많이 씀**)
- `flex-start` / `flex-end`

#### 👉 `flex-wrap` (줄 바꿈 여부)
- `nowrap` (기본): 안 바꿈 (공간 부족하면 찌그러짐)
- `wrap`: 공간 부족하면 다음 줄로 넘김

#### 👉 `align-content` (여러 줄 정렬)
- **조건**: `flex-wrap: wrap` 상태이고, **두 줄 이상일 때만 작동!**
- **역할**: 뭉텅이(여러 줄) 사이의 간격을 조절. (`space-between` 등)


### [ 자식(Item)에게 주는 명령 ]
- `order`: 순서 바꾸기 (기본 0, 숫자 클수록 뒤로 감)
- `flex-grow`: 남는 공간 나눠 갖기 (비율 설정)

## 4️⃣ 반응형 웹 (Media Query)

### 📱 화면 크기에 따라 스타일을 바꿔라!
```css
@media (max-width: 600px) {
    .box { background-color: blue; } 
}
```
> "화면 폭이 600px 이하가 되면, 배경을 파랗게 해라."

## 5️⃣ 심화: Grid & Bootstrap

### ▦ Grid Layout (2차원 배치)
- Flex가 '한 줄'이라면, Grid는 **'바둑판'**
- `grid-template-columns: 1fr 1fr;` (화면을 반반 쪼개기)

### 🅱 Bootstrap (UI 라이브러리)
- 남이 만들어둔 CSS 클래스 모음집.
- 파이썬, 자바, Node.js 상관없이 HTML 쓰는 곳이면 다 씀.
- `class="btn btn-primary"`만 쓰면 예쁜 파란 버튼 뚝딱!
