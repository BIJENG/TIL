# 📘 HTML/CSS 핵심 개념 정리 (Part 1)

---

## 1. HTML (HyperText Markup Language)
웹 페이지의 **뼈대(구조)**를 만드는 언어

### 🏗️ 기본 구조 (Skeleton)
```html
<!DOCTYPE html> <!-- HTML5 문서 선언 -->
<html>
  <head>
    <!-- 메타 정보, 제목, CSS 연결 등 (화면에 안 보임) -->
    <title>My Page</title>
  </head>
  <body>
    <!-- 실제 브라우저 화면에 보이는 내용 -->
    <h1>Hello World</h1>
  </body>
</html>
```

### 🌳 DOM (Document Object Model)
* **개념:** 브라우저가 HTML 코드를 해석(Parsing)하여 만든 **트리 구조의 객체 모델**
* **역할:** 자바스크립트가 HTML을 조작할 수 있게 해주는 인터페이스 (HTML = 설계도, DOM = 실제 건물)

### 🏷️ 주요 태그 및 속성
* **Block 요소:** 한 줄을 다 차지함 (`div`, `p`, `h1~h6`, `form`, `ul`, `li`, `table`)
* **Inline 요소:** 콘텐츠 크기만큼만 차지함 (`span`, `a`, `img`, `input`)
* **Semantic 태그:** `div` 대신 의미를 부여한 태그 (`header`, `main`, `footer`, `nav`) → 검색엔진 최적화(SEO)와 가독성에 좋음
* **입력 폼 (`form`):**
    * `action`: 데이터 보낼 주소
    * `method`: `GET`(조회/URL노출) vs `POST`(전송/숨김)
    * **정규식(Regex):** `pattern` 속성을 이용해 입력값 검증 (예: 전화번호 형식 강제)

## 2. CSS (Cascading Style Sheets)
HTML 요소를 **꾸며주고 배치**하는 언어

### 🔌 CSS 적용 방식 3가지
1.  **Inline Style:** 태그에 직접 작성 (우선순위 높음, 유지보수 나쁨)
    * `<h1 style="color: red;">`
2.  **Internal CSS:** HTML `<head>` 안 `<style>` 태그에 작성
3.  **External CSS (권장):** 별도 `.css` 파일로 분리 후 연결
    * `<link rel="stylesheet" href="style.css">`

### 🎯 선택자 (Selectors) 우선순위 & 종류
* **우선순위(명시도):** `!important` > Inline Style > ID `#` > Class `.` > Tag
* **결합자 (Combinators):**
    * `div p` (자손): div 안의 모든 p
    * `div > p` (자식): div 바로 밑의 p
    * `div + p` (인접 형제): div 바로 뒤에 오는 p 하나
    * `div ~ p` (일반 형제): div 뒤에 오는 모든 형제 p
* **속성 선택자:** `input[type="text"]` (특정 속성을 가진 요소)
* **가상 클래스:** `:hover` (마우스 올렸을 때), `:nth-child(n)` (n번째 자식)

## 3. 📦 박스 모델 (Box Model) - ⭐중요
모든 HTML 요소는 사각형 박스로 이루어져 있다.

| 구성 요소 | 설명 |
| :--- | :--- |
| **Content** | 텍스트나 이미지가 들어있는 실질적 내용 |
| **Padding** | 내용(Content)과 테두리(Border) 사이의 **안쪽 여백** |
| **Border** | 박스의 테두리 |
| **Margin** | 테두리(Border) 바깥의 **외부 여백** (요소 간 간격) |

### ⚠️ Box-Sizing 주의점
기본적으로 `width`를 주면 **Content** 크기만 설정됨. Padding과 Border가 추가되면 박스가 예상보다 커짐.
이를 방지하기 위해 실무에서는 아래 코드를 기본으로 깔고 시작함.

```css
/* 테두리(Border)까지 포함해서 크기 계산해라! */
* {
  box-sizing: border-box;
}
```

## 4. 타이포그래피 & 기타 스타일
* `font-size`: 글자 크기 (`px`, `rem`, `em`)
* `font-weight`: 글자 굵기 (`bold`, `100`~`900`)
* `text-align`: 텍스트 정렬 (`left`, `center`, `right`, `justify`)
* `color` vs `background-color`: 글자색 vs 배경색
* `text-decoration: none`: `a` 태그 밑줄 제거 시 사용
