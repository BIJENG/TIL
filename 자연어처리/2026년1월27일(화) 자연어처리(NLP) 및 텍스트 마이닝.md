# 📘 자연어처리(NLP) 및 텍스트 마이닝 학습 요약

## 1. AI 발전 트렌드와 NLP
* **컴퓨터 비전 (Computer Vision):** 이미지를 처리하는 기술. 이미 사람의 인식 능력을 뛰어넘음.
* **자연어 처리 (NLP):** 인간의 언어를 처리하는 기술. 과거엔 사람보다 못했으나, **ChatGPT(LLM)** 등장 이후 사람 수준에 근접하거나 능가하는 중.
* **멀티모달 (Multimodal):** 텍스트와 이미지를 동시에 이해하고 처리하는 기술 (예: 사진을 보고 설명글 생성).

---

## 2. 텍스트 마이닝 (Text Mining) 개요
### 정의
* 비정형 데이터(텍스트)에서 유의미한 정보나 패턴을 추출하여 정형 데이터(표/숫자)로 바꾸고 분석하는 기법.
* **핵심:** 인간의 언어를 컴퓨터가 이해할 수 있는 언어(Python 등)와 숫자(행렬)로 변환하는 것.

### 분석 3단계 프로세스
`데이터 수집` -> `데이터 전처리` -> `데이터 분석`

---

## 3. 단계별 상세 내용

### ① 데이터 수집 (Data Collection)
* **웹 크롤링 (Web Crawling):** 인터넷상의 데이터를 긁어모으는 기술.
    * **파이썬 라이브러리:** `BeautifulSoup`, `Selenium` 등. (자유도 높음, 코딩 필요)
    * **SaaS 서비스:** 코딩 없이 크롤링해 주는 툴. (편리함, 수집 채널 제한)
* **공공 데이터:** 빅카인즈(뉴스), RISS(논문), 공공데이터포털 등 활용.

### ② 데이터 전처리 (Preprocessing) - "가장 중요한 단계"
컴퓨터가 이해할 수 있게 데이터를 다듬는 과정.
1.  **클렌징:** HTML 태그 제거, 특수문자 제거.
2.  **토큰화 (Tokenization):** 문장을 분석 단위(형태소)로 쪼개기.
    * *도구(Python):* **KoNLPy** (Kkma, Okt, Mecab 등)
    * *예시:* "가방에 들어간다" -> `가방(NNG)`, `에(JKM)`, `들어가(VV)`...
3.  **불용어 처리 (Stopwords):** 분석에 불필요한 단어(은, 는, 이, 가, 그리고 등) 제거.
4.  **정규화 (Normalization):** 어간 추출(Stemming), 표제어 추출(Lemmatization) 등을 통해 단어의 원형 복원 (Go, Goes, Going -> Go).

### ③ 데이터 분석 (Analysis) - 핵심 기법

#### 1) 단어 표현 (Vectorization)
글자를 숫자로 바꾸는 방법.
* **BoW (Bag of Words):** 순서 무시, 단어의 출현 빈도만 기록.
* **TDM (단어문서행렬):** 문서를 행(Row), 단어를 열(Column)로 만들어 빈도수를 채운 표.
    * *단점:* 문서가 많아지면 0이 너무 많은 **희소 행렬(Sparse Matrix)**이 됨.

#### 2) 가중치 계산 (TF-IDF)
단어의 빈도(TF)와 문서의 빈도(DF)를 이용해 **"진짜 중요한 단어"**를 찾는 공식.
* **TF (Term Frequency):** 특정 문서에 자주 등장하면 중요함. (↑)
* **IDF (Inverse Document Frequency):** 모든 문서에 흔하게 등장하면 중요하지 않음. (↓)
* **결론:** 특정 문서에만 자주 등장하는 단어가 핵심 키워드다.

#### 3) 문맥 파악 (N-gram)
* 단어를 n개씩 묶어서 분석하는 기법.
* *예:* "가지 마" (Unigram: 가지, 마 / Bigram: 가지 마)
* 문맥의 의미를 파악하는 데 도움을 줌.

## 4. 고급 분석 및 머신러닝/딥러닝 개념

### 유사도 측정 (Similarity)
* **코사인 유사도 (Cosine Similarity):** 두 문서(벡터) 사이의 **각도**를 측정.
    * 문서의 길이가 달라도 주제가 같으면 유사하다고 판단 (텍스트 분석에서 가장 많이 사용).

### 분류 및 군집화 (Classification & Clustering)
* **나이브 베이즈 (Naive Bayes):** 확률 기반 분류기. (스팸 메일 필터링의 시초)
* **토픽 모델링 (LDA):** 문서 집합에 숨겨진 추상적인 '주제(Topic)'를 확률적으로 찾아내는 기법.

### 네트워크 분석 (Network Analysis)
* **의미 연결망:** 단어들이 동시에 등장(Co-occurrence)하는 관계를 시각화.
* **중심성 (Centrality):** 어떤 단어가 네트워크의 핵심인가?
* **CONCOR 분석:** 단어들의 관계 패턴을 분석하여 **비슷한 성격의 단어군(Cluster)**을 묶어주는 기법.

## 5. AI 분야 포함 관계 (Hierarchy)
**AI ⊃ Machine Learning ⊃ Deep Learning**

1.  **AI (인공지능):** 가장 큰 범주.
2.  **Machine Learning (머신러닝):** 데이터를 통해 학습하는 알고리즘.
    * *포함:* 나이브 베이즈, TF-IDF, 의사결정나무 등 (전통적 NLP).
3.  **Deep Learning (딥러닝):** 인간의 뉴런을 모방한 인공신경망 사용.
    * *포함:* RNN, LSTM, **Transformer(BERT, GPT)** 등 (최신 NLP).
    * *특징:* 데이터가 아주 많아야 하고, 스스로 특징을 학습함.
