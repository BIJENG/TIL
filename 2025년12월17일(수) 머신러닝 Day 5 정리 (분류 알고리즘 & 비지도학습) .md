# 📘 머신러닝 Day 5 정리 (분류 알고리즘 & 비지도학습)

## 1. 분류 알고리즘 전체 지도 (기초 과정 기준)

📌 **분류 알고리즘 = 정답(label)이 있는 지도학습**

---

### 1️⃣ KNN (K-Nearest Neighbors)
- ✔ 분류 알고리즘
- ✔ 거리 기반 알고리즘
- 새로운 데이터 기준으로 **가까운 k개 데이터**를 찾아 다수결로 결정
- ❗ 거리 기반이므로 **스케일링 필수**
- ❌ 설명력 낮음, 데이터 많아지면 느려짐

👉 **개념 이해용 대표 모델**

---

### 2️⃣ 로지스틱 회귀 (Logistic Regression)
- ❗ 이름에 회귀가 들어가지만 **분류 알고리즘**
- 출력: 0 또는 1 (확률 기반)
- 시그모이드(Sigmoid) 함수 사용
- 의료·통계 분야에서 매우 중요

**장점**
- ✔ 해석 가능
- ✔ 어떤 변수가 위험에 영향을 주는지 설명 가능

---

### 3️⃣ 결정트리 (Decision Tree)
- ✔ 분류 알고리즘
- 질문을 계속 나누는 구조
- 조건문 기반 (`if ~ else`)
- 시각화 가능

👉 **헬스케어에서 특히 중요**  
(의사가 이해할 수 있음)

---

### 4️⃣ 랜덤 포레스트 (Random Forest)
- ✔ 분류 알고리즘
- 여러 개의 결정트리를 묶은 앙상블 모델
- 과적합 감소
- 성능 ↑, 설명력 ↓

👉 **결정트리 업그레이드판**

---

### 5️⃣ 서포트 벡터 머신 (SVM)
- ✔ 분류 알고리즘
- 클래스 간 **경계(마진)**를 최대한 크게 만드는 방식
- 고차원 데이터에 강함
- ❗ 수학적 개념 많아 기초 과정에서는 개념 위주

---

### 6️⃣ 나이브 베이즈 (Naive Bayes)
- ✔ 분류 알고리즘
- 확률 기반 모델
- 조건부 확률 사용

👉 **스팸메일 분류 단골 모델**

---

## 2. Projection(정사영)이란?

> **고차원 데이터를 특정한 축(저차원 공간)으로 투사하는 것**

- 차원 축소에서 자주 등장
- PCA의 핵심 개념 중 하나
- “데이터를 새로운 축에 눌러 담는다”는 의미

---

## 3. 비지도학습 (Unsupervised Learning)

📌 **정답(label) 없이 데이터 구조를 찾는 학습 방법**

- 대표 예: PCA, K-means
- 결과는 정답이 아니라 **해석 대상**

---

## 4. PCA (Principal Component Analysis, 주성분 분석)

### 📌 정의
**대표적인 비지도학습 기반 차원 축소(변수 추출) 기법**

---

### 🔑 핵심 아이디어
- 데이터의 **분산을 최대한 보존하는 새로운 축**을 찾는다
- 고차원 데이터를 이 축(저차원 공간)으로 **정사영(Projection)**하여 차원 축소
- 기존 변수들의 **선형결합**으로 새로운 변수(주성분) 생성
- 주성분끼리는 서로 **직교**하며 상관관계 = 0

---

### 🎯 목표
- 정보 손실 최소화
- 데이터의 핵심 구조와 패턴 유지
- 중복된 변수 정보 제거

---

### ❗ 중요한 오해 정리
- PCA는 **모델 ❌**
- PCA는 **전처리 / 특성 변환 기법 ⭕**
- 예측이나 분류를 직접 하지는 않음

---

## 5. 주성분 개수는 어떻게 결정할까?

### 방법 1️⃣ 누적 설명 분산 (Cumulative Explained Variance)
- 주성분들의 설명 분산 합이
- 전체 분산의 **일정 비율(예: 70%, 80%, 90%)**을 넘는 지점까지 선택

---

### 방법 2️⃣ Scree Plot (스크리 플롯)
- x축: 주성분 개수
- y축: 설명 분산 비율
- 그래프의 기울기가 **급격히 완만해지는 지점(엘보우 포인트)**에서 선택

📌 Scree Plot은 **평가지표가 아님**
→ 주성분 개수 결정을 돕는 **시각화 도구**

---

## 6. PCA vs K-means 정리

| 구분 | PCA | K-means |
|---|---|---|
| 학습 유형 | 비지도 | 비지도 |
| 역할 | 차원 축소 | 군집 |
| 대상 | 변수(feature) | 데이터(sample) |
| 결과 | 새로운 변수 | 군집 라벨 |
| 목적 | 정보 요약 | 데이터 그룹화 |

## 7. 오늘 핵심 한 줄 요약

- 분류는 **정답이 있는 지도학습**
- KNN은 **거리 기반 분류 모델**
- PCA는 **전처리용 차원 축소 기법**
- Projection은 **고차원 → 저차원 투사**
- Scree Plot은 **주성분 개수 결정용 시각화**
- 비지도학습 결과는 **정답이 아니라 해석 대상**
