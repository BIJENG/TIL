1. 함수란?

함수는 기능들의 묶음으로 생각하면 편함.

수학에서의 함수: 입력과 출력의 상관관계에 초점

프로그래밍에서의 함수: 재사용성에 방점

장점

코드 재사용 가능

코드 구조화 및 가독성 향상

유지보수 용이

2. 함수 정의와 호출

정의: def 키워드를 사용하여 선언

호출: 정의된 함수를 실행

def greet_patient():
    print("안녕하세요, 환자님!")
    print("오늘 진료에 오신 것을 환영합니다.")

greet_patient()  # 함수 호출

3. 매개변수(Parameter)와 인수(Argument)

매개변수(Parameter): 함수 정의 시 입력값을 받기 위한 변수

인수(Argument): 함수 호출 시 실제로 전달하는 값

# 매개변수: name, age, department
def register_patient(name, age, department):
    print(f"환자 등록: {name}님 ({age}세)")
    print(f"담당 과: {department}")
    print("등록 완료")

# 인수: "김환자", 45, "내과"
register_patient("김환자", 45, "내과")

4. 반환값(return)

return은 함수에서 계산된 결과를 호출한 곳으로 돌려주거나 함수 실행을 종료

두 가지 용법

값 반환

def calculate_bmi(height_cm, weight_kg):
    """BMI 계산"""
    return weight_kg / (height_cm / 100) ** 2


반환 후 다른 코드에서 사용 가능

bmi = calculate_bmi(170, 65)
print(bmi)

5. 기본 매개변수(Default Parameter)

함수 호출 시 인수를 생략하면 기본값 사용

def schedule_appointment(patient_name, doctor="담당의 미정", time="시간 미정", room="진료실 미정"):
    print(f"{patient_name} 환자")
    print(f"담당의: {doctor}, 시간: {time}, 진료실: {room}")

schedule_appointment("김환자")
schedule_appointment("이환자", doctor="홍길동", time="10시")

6. 키워드 인수(Keyword Arguments)

매개변수 이름으로 전달 → 순서 상관 없음

def create_prescription(patient_name, medicine, dosage, frequency):
    print(f"환자: {patient_name}, 약물: {medicine}, 용량: {dosage}, 횟수: {frequency}")

create_prescription(
    frequency="1일 3회",
    patient_name="김환자",
    dosage="500mg",
    medicine="아스피린"
)

# 일부 위치, 일부 키워드 가능
create_prescription("이환자", "타이레놀", dosage="250mg", frequency="1일 2회")


⚠️ 주의: 키워드 인수 사용 시작 후에는 모든 매개변수를 키워드로 명시해야 함

7. 함수 설계 원칙

단일 책임: 하나의 함수는 하나의 기능만

의미 있는 함수명: 이름만 봐도 기능 알 수 있게

적절한 크기: 일반적으로 20줄 이내

재사용성: 다양한 상황에서 활용 가능

독립성: 다른 함수에 과도하게 의존하지 않게

# ❌ 나쁜 예: 여러 기능 한 번에
def process_patient_data(name, age, height, weight):
    bmi = weight / (height / 100) ** 2
    print(f"환자명: {name}, 나이: {age}")
    if bmi >= 25: prescription = "식이요법 처방"
    else: prescription = "정기검진"
    return bmi, prescription

# ✅ 좋은 예: 기능별 분리
def calculate_bmi(height, weight):
    return weight / (height / 100) ** 2

def classify_weight_status(bmi):
    if bmi < 18.5: return "저체중"
    elif bmi < 25: return "정상체중"
    else: return "과체중"

def print_patient_info(name, age):
    print(f"환자명: {name}, 나이: {age}")

def generate_prescription(weight_status):
    return "식이요법 처방" if weight_status == "과체중" else "정기검진"

8. 타입 힌팅(Type Hints)

함수 입출력 타입 명시 → 가독성, 안정성 향상

from typing import Tuple

def check_appointment_availability(hour: int) -> Tuple[bool, str]:
    return True, "AVAILABLE"

def add_appointment(patient_name: str, hour: int) -> str:
    return f"{patient_name} - {hour}시: 예약 완료"


장점

코드 가독성 향상

버그 예방

IDE 자동완성 및 오류 검출

문서화 효과

팀 협업 용이

⚠️ 타입 힌팅은 Python 실행에 영향 없음
